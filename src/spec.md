# Specification

## Summary
**Goal:** Create a personal portfolio website for Meghana Jha, a Biomedical Science student at University of Delhi, showcasing her academic profile, projects, skills, and contact information with photo/certificate upload functionality.

**Planned changes:**
- Create homepage with hero section displaying name and academic program
- Add Profile section with bio about biomedical science focus
- Add Contact section with phone and email (clickable mailto links)
- Add Education section showing three educational entries (BSc, Class XII, Class X) with grades and achievement note
- Add Projects & Internships section listing student activities, EduThon 2025 win, hospital internship, and presentations
- Add Skills section displaying 11 professional and technical skills
- Add Hobbies section showing 6 personal interests
- Add Languages section with proficiency levels (English, Hindi, Sanskrit)
- Add References section with three professors' contact information (clickable mailto links)
- Implement backend file upload system for photos and certificates with metadata storage
- Create frontend media gallery with upload capability for photos and certificates
- Apply academic/professional design with sophisticated color palette (excluding blue and purple), clean typography, and research-focused layout
- Store generated profile photo and certificate background as static assets in frontend/public/assets/generated

**User-visible outcome:** A professional portfolio website displaying Meghana Jha's complete academic profile with all sections clearly organized, allowing visitors to view her credentials and contact her, plus the ability to upload and display photos and certificates in a gallery format.
